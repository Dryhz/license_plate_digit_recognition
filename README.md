# 基于结构先验与混合特征的极小样本车牌数字识别系统

## 一、项目概述

本项目是一个基于计算机视觉和机器学习的车牌数字识别系统，主要实现了从车牌图像中定位、分割并识别数字的完整流程。

## 三、核心技术架构

### 3.1 系统流程

```
输入图像 → 车牌定位 → 字符分割 → 特征提取 → 数字识别 → 后处理 → 输出结果
```

### 3.2 关键模块

| 模块 | 功能描述 | 核心技术 |
|------|----------|----------|
| **车牌定位** | 检测图像中的车牌区域 | 颜色阈值（HSV）+ 边缘检测（Sobel） |
| **字符分割** | 从车牌区域中提取单个字符 | Otsu阈值 + 形态学操作 + 连通分量分析 |
| **特征提取** | 将字符图像转换为特征向量 | HOG特征 + 投影特征 + 几何特征 |
| **数字识别** | 识别字符并分类为数字 | SVM分类器 + 模板匹配 + 结构规则 |

## 四、算法实现细节

### 4.1 车牌定位

支持两种定位策略：

1. **基于颜色的定位**（蓝色车牌）：
   - 使用HSV颜色空间检测蓝色区域
   - 通过形态学闭运算连接区域
   - 筛选宽高比约为3.2的矩形区域

2. **基于边缘的定位**：
   - Sobel算子提取水平边缘
   - 形态学闭运算形成候选区域
   - 作为颜色定位的补充

### 4.2 字符分割

采用多策略阈值分割：

- **蓝色车牌**：Otsu阈值 + Top-hat变换 + HSV白色区域 + 自适应阈值
- **浅色车牌**：HSV暗色区域 + Otsu反阈值

选择标准基于：字符数量、宽度均匀性、高度占比、水平分布范围

### 4.3 特征提取

每个字符提取426维特征向量：

| 特征类型 | 维度 | 描述 |
|----------|------|------|
| HOG特征 | 324 | 方向梯度直方图 |
| 行投影 | 16 | 垂直方向像素分布 |
| 列投影 | 16 | 水平方向像素分布 |
| 几何特征 | 2 | 密度、宽高比 |

### 4.4 数字识别（多策略融合）

```
SVM分类器（60%权重）
        ↘
    加权融合 → 结构规则修正 → 最终识别结果
        ↗
模板匹配（40%权重）
```

**结构规则修正示例**：
- `0 → 9`：当右侧有尾部且左下区域空白时
- `8 → 3`：当左侧中间区域空白且3的模板匹配度高时
- `5/0/8 → 6`：当左侧有环形且6的匹配度高时

## 五、数据集与数据增强

### 5.1 模板数据集

使用10个数字模板（0-9），通过数据增强生成训练集：

| 变换类型 | 参数范围 |
|----------|----------|
| 旋转角度 | [-7°, 7°] |
| 缩放比例 | [0.88, 1.12] |
| 水平平移 | [-3, 3] |
| 垂直平移 | [-4, 4] |
| 剪切变换 | [-0.06, 0.06] |
| 高斯模糊 | 3x3核（20%概率） |
| 腐蚀/膨胀 | 2x2核（各10%概率） |

### 5.2 增强规模

每个模板生成90个增强样本，共900个训练样本。

## 六、模型选择与评估

### 6.1 模型配置对比

| 模型配置 | 核函数 | 参数 | 验证准确率 |
|----------|--------|------|------------|
| linear_C0.5 | 线性 | C=0.5 | ~99% |

| 识别阶段 | 准确率 |
|----------|--------|
| 仅模板匹配 | ~85% |
| 仅SVM | ~92% |
| SVM + 模板融合 | ~96% |
| + 结构规则修正 | **~98%** |

## 十、项目亮点

1. **多策略融合识别**：结合机器学习（SVM）、传统模板匹配和结构规则，提升鲁棒性
2. **自适应阈值选择**：根据车牌类型自动选择最优分割策略
3. **数据增强**：通过仿射变换扩展训练样本，增强模型泛化能力
4. **可视化输出**：生成多种分析图表，便于结果分析和论文写作
5. **完整的实验框架**：包含超参数搜索、消融实验、性能评估等完整流程
