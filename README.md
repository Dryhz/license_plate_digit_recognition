# 基于结构先验与混合特征的极小样本车牌数字识别系统

## 一、摘要

针对智能交通系统（ITS）中特定场景下面临的极小样本车牌数字识别任务，本文提出了一套高度可解释、完全可复现的多阶段混合识别框架。在缺乏大规模标注数据的受限条件下，传统的深度学习端到端模型往往面临严重的过拟合风险，且缺乏预测过程的透明度。

为此，本文首先提出了一种结合颜色空间先验与横向局部梯度密集性的自适应车牌定位算法，随后根据车牌底色分布动态选择二值化策略以完成高精度的字符分割。在特征提取与识别阶段，本研究在仅有10张标准数字模板的极端条件下，通过可控仿射与形态学扰动将训练集扩增至910个样本，并创新性地提出融合方向梯度直方图（HOG）、行列投影分布、前景密度及宽高比的结构化联合特征。利用支持向量机（SVM）进行高维空间的大间隔分类后，本文进一步引入基于归一化模板相关度和局部笔画密度分析的结构后处理模块，有效抑制了在小样本下极易发生的0/9、3/8、5/6等相似字形的混淆现象。

实验结果表明，在经过严格的脚本核验后，该系统在纯净样本上数字识别率达到100.0%；在包含复杂干扰（强反光、背景招牌、尺度变化）的仿真样本上，数字定位F1分数与仿真样本目标数字端到端识别准确率均达到83.33%。横跨全数据集的总体目标数字准确率达到了93.75%（15/16）。相较于单一的模板匹配方法，本文方法在逻辑可解释性、复杂背景抑制及结果可追溯性上展现出显著优势，并对仿真干扰下的单字拓扑失效进行了深刻的错误根源剖析，为小样本条件下的车牌字符识别提供了一种兼具工程鲁棒性与科研严谨性的完整解决方案。

**关键词**  自动车牌识别；极小样本学习；方向梯度直方图；支持向量机；自适应字符分割；结构启发后处理

## 三、核心技术架构

### 3.1 系统流程

```
输入图像 → 车牌定位 → 字符分割 → 特征提取 → 数字识别 → 后处理 → 输出结果
```

### 3.2 关键模块

| 模块 | 功能描述 | 核心技术 |
|------|----------|----------|
| **车牌定位** | 检测图像中的车牌区域 | 颜色阈值（HSV）+ 边缘检测（Sobel） |
| **字符分割** | 从车牌区域中提取单个字符 | Otsu阈值 + 形态学操作 + 连通分量分析 |
| **特征提取** | 将字符图像转换为特征向量 | HOG特征 + 投影特征 + 几何特征 |
| **数字识别** | 识别字符并分类为数字 | SVM分类器 + 模板匹配 + 结构规则 |

## 五、项目亮点

1. **多策略融合识别**：结合机器学习（SVM）、传统模板匹配和结构规则，提升鲁棒性
2. **自适应阈值选择**：根据车牌类型自动选择最优分割策略
3. **数据增强**：通过仿射变换扩展训练样本，增强模型泛化能力
4. **可视化输出**：生成多种分析图表，便于结果分析和论文写作
5. **完整的实验框架**：包含超参数搜索、消融实验、性能评估等完整流程
